Search syntax

Combine any of the below. Terms are ANDed together unless you write OR; prefix a term with - to negate it. Click any example to run it.

Text Fields Numbers Keywords Format Sets Combining

Plain text

Bare words match the card name and rules text. Wrap a phrase in quotes for an exact match, or prefix it with ! to exclude it.

honor "Doji Hoturi" !"unique"

Fields

Filter by a field with field:value. Available fields: name: type: clan: deck: text: set: rarity:.

Most have a one-letter shortcut: t: type, c: clan, o: text, s: set, r: rarity, f: force.

type:holding c:Crane t:personality o:cavalry

Numbers

Compare any stat with : (equals), >, >=, <, or <=.

Stats: force chi gold focus ph (personal honor) province honor_requirement starting_honor.

force>=4 chi>=2 gold<=3 ph>=3 clan:Lion

Keywords & traits

is: (or has:) matches any keyword or trait — is:shugenja, is:shadowlands, is:cavalry, is:unique, and the rest.

Combine inside one term: is:a|b for either, is:a&b for both.

is:shugenja is:shadowlands is:cavalry|naval is:unique force>=5

Format (legality)

Cards legal in a format, by short arc name or full name — format:diamond is the same as format:"Rain of Blood (Diamond)".

Aliases: imperial jade gold diamond lotus samurai celestial emperor ivory onyx shattered · modern legacy. Compare along the timeline with > >= < <=.

format:diamond format>celestial format>=gold format<=diamond

Sets

A set by its code or full name — set:GE is the same as set:"Gold Edition".

Compare by release date for ranges: set>=GE, set<=DE, or both for a window.

set:GE set:"Gold Edition" is:unique set>=GE set<=DE

Combining

Stack terms to narrow a search; everything is ANDed. Negate with -, or branch with OR.

clan:Crane OR clan:Lion -type:event is:shugenja c:Crab f>=4 format:diamond
